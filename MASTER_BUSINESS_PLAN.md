|---|---|---|

| **Uptime SLA** | 99.9% | Single point of failure (local machine) eliminated by Phase 1 |
| **Hallucination-free** | KB = only source of truth | If confidence <70%, agent asks a clarifying question or escalates |
| **Human override** | <30 seconds | Any agent conversation transferable to human within 30 seconds |

## SECTION 07 — ADR: ARCHITECTURE DECISION RECORD

Every major technical bet, justified with an escape hatch.

| # | Decision | Rationale | Alternative rejected | Escape hatch |
|---|---|---|---|---|
| **ADR-001** | Claude (Anthropic) as AI backbone | Best multilingual DE/EN/ES generation. Anthropic DPA = GDPR-native. Zero training on customer data. | OpenAI GPT-4.1 (GDPR risk) · Llama local (accuracy gap) | Fallback to GPT-4.1 via single env variable. Logic is provider-agnostic. |
| **ADR-002** | MCP as integration layer | Standardized protocol for agent↔tool. Connects Blotato, Adspirer, Google, WhatsApp without custom code. | Custom REST (10× maintenance) · n8n (infra complexity at Phase 0) | Each MCP server independently replaceable. Core logic doesn't change. |
| **ADR-003** | Python + Flask → FastAPI | Python is AI lingua franca. Flask for Phase 0. FastAPI for Phase 1+ (async, 10× faster under load). | Node.js (team is Python-first) · LangChain (obscures what agents do) | FastAPI is drop-in Flask replacement. Migration is incremental. |
| **ADR-004** | SQLite → PostgreSQL (Supabase) | SQLite for Phase 0: zero setup. Supabase for Phase 1+: managed PostgreSQL, row-level security, multitenant. | MongoDB (schemaless liability for financial data) · DynamoDB (AWS lock-in, not GDPR-default) | SQLite schema mirrors PostgreSQL. One-day migration. |
| **ADR-005** | Hetzner (Germany) for VPS | Data stays in Germany. GDPR by geography. €5–20/month for production VPS. | AWS Frankfurt (10× cost) · Vercel (not suitable for long-running Python agents) | Docker Compose. Any EU-based VPS = 2-hour swap. |

### The portable logic rule

> **All business logic lives in plain Python.** No framework dependency, no vendor lock-in at the logic layer. The knowledge base is a Python dict. The intent classifier is regex + LLM. The escalation rules are if/elif. If Anthropic doubles prices tomorrow, we switch providers in one day. The agents are more durable than any tool they use.
